# Examples

Real generated output from `tool-docs generate` + `tool-docs distill`.

Each directory is the full output for one tool: a compressed `SKILL.md` (~1â€“2KB) and a `docs/` subfolder with extended reference, recipes, and troubleshooting.

| Tool | SKILL.md | What it shows |
|------|----------|---------------|
| [`jq/`](jq/) | [SKILL.md](jq/SKILL.md) | JSON filtering/transformation |
| [`curl/`](curl/) | [SKILL.md](curl/SKILL.md) | HTTP requests, complex flag set |
| [`gh/`](gh/) | [SKILL.md](gh/SKILL.md) | GitHub CLI, subcommand-heavy tool |
| [`ffmpeg/`](ffmpeg/) | [SKILL.md](ffmpeg/SKILL.md) | Massive `--help`, shows compression |
| [`uv/`](uv/) | [SKILL.md](uv/SKILL.md) | Python package manager |

These files are generated by running:

```bash
tool-docs generate --only jq curl gh ffmpeg uv
tool-docs distill  --only jq curl gh ffmpeg uv
```

The source `--help` output and raw parsed JSON live in `~/.agents/docs/tool-docs/<tool-id>/` on your machine after running `generate`.
